# Taoglas Template Quick Reference

## COLORS
| Element | Hex Code |
|---------|----------|
| Gold (borders/accents) | `#F1C232` |
| Black (headers/text) | `#000000` |
| White (text on black) | `#FFFFFF` |
| Links | `#1155CC` |

## FONTS
| Use | Font | Weight | Size |
|-----|------|--------|------|
| Header Title | Plus Jakarta Sans | Bold | 20pt |
| H2 Section Title | Rubik | Bold | 17pt |
| H3 Subsection | Rubik | Bold | 13pt |
| Section Label (gold) | Rubik | Normal | 12pt |
| Body Text | Rubik | Normal | 10pt |
| Bullets | Rubik | Normal | 10pt |

## PAGE SETUP
- **Size:** 8.5" × 11" (Letter)
- **Margins:** 1" all sides
- **Header/Footer:** 0.5"
